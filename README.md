# Métricas de Avaliação em Classificação

Neste projeto são calculadas as principais métricas de avaliação de modelos de classificação: acurácia, sensibilidade (recall), especificidade, precisão e F-score.

Foi utilizado um dataset de doenças cardiovasculares, treinado com regressão logística. A partir do modelo, foi gerada uma matriz de confusão, usada para obter os valores de VP, VN, FP e FN.

O objetivo do projeto é entender como cada métrica funciona e como é calculada, e não otimizar o desempenho do modelo.
